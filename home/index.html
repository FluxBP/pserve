PermaServe

This server is running PermaServe, a simple NodeJS application that reads data stored in the consensus memory (RAM) of an Antelope blockchain using the PermaStore standard.

This installation is connected to the UX Network blockchain, and is using the PermaStore contract deployed to the permastoreux smart contract account.

Instructions:

To browse all cached PermaStore pages, navigate to the root URL.

To browse a specific PermaStore page, navigate to the top-level directory named after that page:

  • Visit page named example: /example

If a requested page is not already cached, PermaServe will attempt to retrieve it from the blockchain.

The page cache can be explored by visiting /_pages

The node cache can be explored by visiting /_nodes
